A template for a class

```py
def Class():
    def __init__(self, attribute1, attribute2):
        self.attribute1 = attribute1
        self.attribute2 = attribute2

    def swap(self):
        temp = self.attribute1
        self.attribute2 = attribute1
        self.attribute1 = temp 
```

```py
Class('attribute1', 'attribute2') #Class object
```
